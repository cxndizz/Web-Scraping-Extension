# BrowserHarvest: เครื่องมือดึงข้อมูลเว็บอัตโนมัติ

![BrowserHarvest Logo](assets/logo.png)

BrowserHarvest เป็นแอปพลิเคชันที่ช่วยให้คุณดึงข้อมูลจากเว็บไซต์ได้อย่างง่ายดายโดยการเลือกองค์ประกอบบนหน้าเว็บเพียงไม่กี่คลิกเท่านั้น

## ฟีเจอร์หลัก

- **เปิดเว็บในแอพ**: วาง URL และเปิด browser ในแอพได้ทันที
- **คลิกเลือกแล้วจำองค์ประกอบ**: เลือกองค์ประกอบเพียงครั้งเดียว แล้วระบบจะจำและดึงข้อมูลในรูปแบบเดียวกันจากทุกรายการ
- **ทำงานทีละหน้า หรือข้ามหน้า**: เลือกเก็บข้อมูลทั้งหน้าเดียวหรือข้ามหลายหน้าโดยเลือกปุ่มหน้าถัดไป
- **จำกัดจำนวน records**: กำหนดจำนวนข้อมูลที่ต้องการเก็บได้
- **ส่งออกข้อมูลหลายรูปแบบ**: ส่งออกเป็น CSV, JSON, SQL ได้อย่างง่ายดาย

## การติดตั้ง

### วิธีที่ 1: ดาวน์โหลดไฟล์ติดตั้ง

1. ดาวน์โหลด [ไฟล์ติดตั้งล่าสุด](https://github.com/example/browserharvest/releases/latest) สำหรับระบบปฏิบัติการของคุณ
2. เปิดไฟล์และทำตามขั้นตอนการติดตั้ง

### วิธีที่ 2: ติดตั้งจากซอร์สโค้ด

1. โคลนหรือดาวน์โหลดโค้ด
```
git clone https://github.com/example/browserharvest.git
cd browserharvest
```

2. ติดตั้ง dependencies
```
npm install
```

3. รันแอป
```
npm start
```

## วิธีการใช้งาน

1. **กรอก URL เป้าหมาย**
   - ใส่ URL ของเว็บไซต์ที่ต้องการดึงข้อมูล
   - คลิกปุ่ม "เปิดเว็บไซต์" เพื่อเปิด browser

2. **เลือกองค์ประกอบ**
   - คลิกปุ่ม "เลือกองค์ประกอบสินค้า" เพื่อเริ่มเลือก
   - คลิกที่องค์ประกอบบนหน้าเว็บที่ต้องการดึงข้อมูล (เช่น ชื่อสินค้า ราคา รูปภาพ)
   - ตั้งชื่อฟิลด์และเลือกคุณสมบัติที่ต้องการ (เช่น text, href, src)
   - เพิ่มฟิลด์เพิ่มเติมตามต้องการ

3. **ตั้งค่าการเลื่อนหน้า** (ถ้าต้องการเก็บหลายหน้า)
   - คลิกปุ่ม "เลือกปุ่มหน้าถัดไป" และคลิกที่ปุ่มเลื่อนหน้าบนเว็บไซต์
   - ระบุจำนวนหน้าสูงสุดที่ต้องการเก็บข้อมูล
   - ระบุจำนวน records สูงสุดที่ต้องการ

4. **เริ่มเก็บข้อมูล**
   - คลิกปุ่ม "เริ่มเก็บข้อมูล"
   - ดูความคืบหน้าและสถานะการเก็บข้อมูล

5. **ส่งออกข้อมูล**
   - เลือกรูปแบบส่งออกที่ต้องการ: CSV, JSON, SQL
   - เลือกตำแหน่งที่ต้องการบันทึกไฟล์

## ตัวอย่างการใช้งาน

### ตัวอย่างที่ 1: เก็บรายการสินค้าจากเว็บ E-commerce

1. ใส่ URL เว็บ E-commerce และกดเปิด
2. เลือกองค์ประกอบ:
   - ฟิลด์ "title" - เลือกชื่อสินค้า
   - ฟิลด์ "price" - เลือกราคาสินค้า
   - ฟิลด์ "image" - เลือกรูปสินค้า (attribute: src)
   - ฟิลด์ "link" - เลือกลิงก์ไปยังหน้ารายละเอียด (attribute: href)
3. เลือกปุ่มเลื่อนหน้า ตั้งค่าจำนวนหน้าเป็น 5
4. กดเริ่มเก็บข้อมูล
5. ส่งออกเป็น CSV เพื่อนำไปเปิดใน Excel

### ตัวอย่างที่ 2: เก็บรายการบทความจากเว็บข่าว

1. ใส่ URL เว็บข่าวและกดเปิด
2. เลือกองค์ประกอบ:
   - ฟิลด์ "title" - เลือกหัวข้อข่าว
   - ฟิลด์ "date" - เลือกวันที่โพสต์
   - ฟิลด์ "summary" - เลือกบทคัดย่อ
   - ฟิลด์ "link" - เลือกลิงก์ข่าว (attribute: href)
3. กดเริ่มเก็บข้อมูล
4. ส่งออกเป็น JSON

## ข้อควรระวังและข้อจำกัด

1. **การเคารพกฎหมายและเงื่อนไขการใช้งาน**:
   - ตรวจสอบ robots.txt และเงื่อนไขการใช้งานของเว็บไซต์ก่อนดึงข้อมูล
   - ใช้งานอย่างรับผิดชอบและไม่สร้างภาระให้กับเซิร์ฟเวอร์ของเว็บไซต์

2. **ข้อจำกัดทางเทคนิค**:
   - เว็บไซต์บางแห่งอาจมีการป้องกันการดึงข้อมูลอัตโนมัติ
   - เว็บไซต์ที่โหลดด้วย JavaScript แบบ dynamic อาจทำงานได้ไม่สมบูรณ์

3. **ความเป็นส่วนตัวของข้อมูล**:
   - หลีกเลี่ยงการเก็บข้อมูลส่วนบุคคลที่ละเอียดอ่อน
   - ปฏิบัติตามกฎหมายคุ้มครองข้อมูลส่วนบุคคลที่เกี่ยวข้อง

## การแก้ไขปัญหาเบื้องต้น

1. **Browser ไม่เปิดหรือไม่ทำงาน**
   - ตรวจสอบว่ามีการติดตั้ง dependencies ครบถ้วน
   - ตรวจสอบว่า Playwright ทำงานได้ถูกต้อง
   - รันคำสั่ง `npx playwright install chromium` เพื่อติดตั้ง browser ใหม่

2. **ไม่สามารถเลือกองค์ประกอบได้**
   - ลองใช้ selector ที่เฉพาะเจาะจงมากขึ้น
   - เว็บไซต์อาจมีโครงสร้าง DOM ที่ซับซ้อน ลองเลือกองค์ประกอบอื่น

3. **การเลื่อนหน้าไม่ทำงาน**
   - ตรวจสอบว่าได้เลือกปุ่มเลื่อนหน้าที่ถูกต้อง
   - บางเว็บไซต์ใช้ AJAX หรือ Infinite Scroll ซึ่งอาจต้องปรับวิธีการทำงาน

## การพัฒนาต่อ

หากคุณต้องการพัฒนาหรือแก้ไขโค้ด BrowserHarvest สามารถทำได้ดังนี้:


ในการสร้างไฟล์ติดตั้งสำหรับแจกจ่าย:
